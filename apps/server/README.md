# Backend for Atsevofni Challenge Overview

This is the repository for Atsevofni challenge's backend, built with FastAPI (Python).

## Quickstart

Prerequisites:

- The [uv](https://docs.astral.sh/uv) package manager
- Python 3.13+

If you don't have [uv](https://docs.astral.sh/uv) already installed, you can follow this [guide](https://docs.astral.sh/uv/getting-started/installation/).

### Starting the dev server

Instructions to run the whole project can be found on the root folder. If you want to run ONLY the backend services, you need to run `pnpm dev:server` on the root folder of the project.

## What about deploying it?

There are many ways to deploy the service, the easiest way is to use the existing `Dockerfile`.
